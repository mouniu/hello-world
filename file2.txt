首先创建Git仓库 （目标文件夹下）
git init

git remote 查看本文件夹有多少个仓库！

远程github仓库如果已经新建过得话可以拉去到本地

git remote add  你的仓库名字 git@github.com:你的远程仓库用户名/你的仓库名字.git

添加好本地的存储仓库之后 
使用 
git pull 你的仓库名字 master  拉去master的分支文件

然后 你可以在目标文件夹下  新建你的任何东西，建好之后 可以使用

git add 你的文件   进行添加到本地仓库

git commit -m "备注"  进行提交




-----------修改后的文件的操作
我们可以修改我们的文件，修改后的文件仍然需要提交的我们的本地仓库，所以我们修改后文件之后可以使用 
git status  查看我们的仓库当前的状态，有哪些被修改了云云。但是虽然被修改了我们不知道修改了具体的内容怎么办使用
git deff 查看我们那些东西被修改了。

之后我们可以提交到本地仓库了，提交修改和提交新文件是一样的操作
git add 修改的文件
git commit -m "备注" 进行提交
